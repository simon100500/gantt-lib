**Логика из примера (перенести 1-в-1, адаптировав под проект):**

1. Добавить импорты из date-fns: `parse, addDays, addMonths, addYears, subDays, subMonths, subYears, startOfDay` (format и isValid уже есть).

2. Новые state/ref внутри DatePicker:
   ```typescript
   const [inputValue, setInputValue] = useState('');        // строка dd.MM.yy
   const dateInputRef = useRef<HTMLInputElement>(null);
   ```

3. Синхронизировать inputValue с prop `value`:
   ```typescript
   useEffect(() => {
     if (value) {
       const d = new Date(value + 'T00:00:00Z');
       if (isValid(d)) setInputValue(format(d, 'dd.MM.yy'));
     } else {
       setInputValue('');
     }
   }, [value]);
   ```

4. При открытии попапа (open=true) — автофокус на инпут:
   ```typescript
   useEffect(() => {
     if (open) {
       setTimeout(() => {
         if (dateInputRef.current) {
           dateInputRef.current.focus();
           selectSegment(0);
         }
       }, 50);
     }
   }, [open]);
   ```

5. Segments и функции сегментной навигации (скопировать из примера):
   ```typescript
   const segments = [
     { start: 0, end: 2, label: 'day', max: 31 },
     { start: 3, end: 5, label: 'month', max: 12 },
     { start: 6, end: 8, label: 'year', max: 99 },
   ];
   const selectSegment = (pos: number) => {
     if (!dateInputRef.current) return;
     const segment = segments.find(s => pos >= s.start && pos <= s.end) || segments[0];
     dateInputRef.current.setSelectionRange(segment.start, segment.end);
   };
   ```

6. handleFocus, handleMouseDown — из примера (выделяют нужный сегмент).

7. updateFromDate — обновляет inputValue И вызывает onChange с ISO-строкой:
   ```typescript
   const updateFromDate = (newDate: Date) => {
     if (!isValid(newDate)) return;
     setInputValue(format(newDate, 'dd.MM.yy'));
     const iso = [
       newDate.getFullYear(),
       String(newDate.getMonth() + 1).padStart(2, '0'),
       String(newDate.getDate()).padStart(2, '0'),
     ].join('-');
     onChange?.(iso);
   };
   ```

8. handleCalendarSelect (вместо handleSelect) — вызывает updateFromDate И закрывает попап:
   ```typescript
   const handleCalendarSelect = (day: Date) => {
     updateFromDate(day);
     setOpen(false);
   };
   ```

9. handleKeyDown для инпута — скопировать из примера полностью (ArrowUp/Down меняют сегмент, ArrowLeft/Right переключают сегменты, цифры вводятся с валидацией). Добавить обработку Escape (закрыть попап) и Enter (закрыть попап).
