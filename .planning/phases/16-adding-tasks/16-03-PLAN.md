New handlers to add:
```typescript
const handleAdd = useCallback((task: Task) => {
  setTasks(prev => [...prev, task]);
}, []);

const handleDelete = useCallback((taskId: string) => {
  setTasks(prev => prev.filter(t => t.id !== taskId));
}, []);
```

Note: The library's GanttChart.handleDelete already calls onChange with the dependency-cleaned array before calling onDelete. So handleDelete in page.tsx only needs to filter by taskId — the dependency cleanup is handled inside the library.
