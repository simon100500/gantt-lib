  <files>packages/gantt-lib/src/utils/dependencyUtils.ts</files>
  <action>
    Add 4 export functions to dependencyUtils.ts:

    1. getChildren(parentId: string, tasks: Task[]): Task[]
       - Filter tasks where task.parentId === parentId
       - Return filtered array (empty if none)

    2. isTaskParent(taskId: string, tasks: Task[]): boolean
       - Use tasks.some(t => t.parentId === taskId)
       - Returns false if no children

    3. computeParentDates(parentId: string, tasks: Task[]): { startDate: Date; endDate: Date }
       - Get children via getChildren()
       - If children.length === 0: find parent task, return own startDate/endDate as Date objects
       - Extract all startDates and endDates as Date objects (new Date(child.startDate))
       - Use Math.min(...startDates.map(d => d.getTime())) for start
       - Use Math.max(...endDates.map(d => d.getTime())) for end
       - Return { startDate: new Date(minTime), endDate: new Date(maxTime) }

    4. computeParentProgress(parentId: string, tasks: Task[]): number
       - Get children via getChildren()
       - If children.length === 0: return 0
       - Initialize totalWeight = 0, weightedSum = 0
       - Use existing DAY_MS constant from dependencyUtils.ts
       - For each child:
         * Calculate duration: (endTime - startTime + DAY_MS) / DAY_MS (inclusive)
         * Get progress: child.progress ?? 0
         * totalWeight += duration
         * weightedSum += duration * progress
       - Return weightedSum / totalWeight (or 0 if totalWeight === 0)
       - Round to 1 decimal: Math.round(result * 10) / 10

    Import Task type from '../types' at top of file.
