# Список изменений

Все значительные изменения проекта будут задокументированы в этом файле.

## [0.21.0] - 2026-03-18

### Возможности
- Добавлены внешние подписи длительности задач - отображаются снаружи бара, когда текст не помещается внутри
- Улучшен UI TaskList: символ "-" для пустого прогресса, суффикс "д" для длительности, жирные даты для родительских задач
- Добавлены примеры кастомных выходных календарей на демо-страницу

### Исправления
- Исправлено позиционирование внешних подписей и обработка переполнения
- Исправлено переполнение текста родительских задач
- Исправлено выравнивание внешних меток с краем бара
- Позиционирование внешних меток относительно визуального края бара с учётом border-radius
- Убраны лишние отступы в rightLabels flex children

## [0.20.0] - 2026-03-18

### Критические изменения
- **Удалены:** пропсы `weekends` и `workdays` - заменены единым массивом `customDays`
- **Новое:** проп `customDays?: CustomDayConfig[]`, где `CustomDayConfig = { date: Date; type: 'weekend' | 'workday' }`
- Миграция: замените `weekends={[date1, date2]}` на `customDays={[{ date: date1, type: 'weekend' }, { date: date2, type: 'weekend' }]}`

### Рефакторинг
- Унификация API кастомных выходных/рабочих дней в единый массив `customDays` с явной типизацией
- Добавлен интерфейс `CustomDayConfig` и утилита `createCustomDayPredicate` с O(1) lookup на основе Set
- Обновлены все компоненты (GanttChart, TaskList, TaskListRow, Calendar, DatePicker) для использования нового API
- Сохранён проп `isWeekend?: (date: Date) => boolean` для кастомной логики предиката

### Возможности
- Добавлен порядок приоритетов для кастомных дней: workday > weekend > базовый предикат > по умолчанию

### Исправления
- Обновлена демо-страница для использования нового API `customDays`

### Тесты
- Добавлено 8 новых модульных тестов для `createCustomDayPredicate`, покрывающих все сценарии

## [0.19.0] - 2026-03-18

### Возможности
- Добавлена поддержка кастомных выходных календаря с предикатом isCustomWeekend
- Добавлена утилита createIsWeekendPredicate для гибких паттернов выходных
- Добавлена утилита createDateKey для поиска по датам
- Добавлен проп isWeekend к компоненту Calendar
- Передача кастомных выходных в DatePicker в TaskList
- Добавлены заголовки будних дней (Пн-Вс) в компонент Calendar

### Исправления
- Использование UTC дат для согласованного определения выходных во всех часовых поясах
- Удалены неиспользуемые списки задач кастомных выходных и обновлены выходные GanttChart

### Рефакторинг
- Улучшены утилиты дат для поддержки кастомных выходных
