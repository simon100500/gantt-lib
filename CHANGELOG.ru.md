## [0.50.0] - 2026-03-30

### Критические изменения

- **Свойство `editor` удалено** — Пользовательские колонки должны использовать `renderEditor` вместо `editor`:
  ```diff
  - editor: ({ task, updateTask, closeEditor }) => ...
  + renderEditor: ({ task, updateTask, closeEditor }) => ...
  ```
- **Изменён путь импорта** — Тип `TaskListColumn` импортируется из корня пакета (`gantt-lib`), не из внутреннего пути `taskListColumns`
- **Только числовая `width`** — Ширина колонки принимает только `number` (пиксели), строковые значения не поддерживаются
- **Размещение через `before`/`after`** — Единственный способ позиционирования пользовательских колонок

### Возможности

- Добавлен проп `additionalColumns` в GanttChart для пользовательских колонок TaskList
- Добавлен обобщённый тип `TaskListColumn<TTask>` с размещением через якоря (`before`/`after`)
- Добавлен конвейер `resolveTaskListColumns` для унифицированного разрешения колонок
- Добавлена фабрика встроенных колонок TaskList
- Добавлен колбэк `onInsertAfter` для вставки задач между строками
- Добавлены демо-компоненты: AdditionalColumnsChart, ConstructionChart

### Рефакторинг

- Унифицирован рендеринг заголовков и строк через конвейер resolvedColumns
- Унифицировано состояние редактора в едином `editingColumnId`
- Удалены приведения `as` Task из цепочки пропов additionalColumns
- Выделены демонстрационные данные в отдельный модуль
- Удалён устаревший редактор и мостовой реэкспорт

### Исправления

- Улучшена типобезопасность колбэков `onTasksChange`
- Исправлена унификация состояния редактора пользовательских колонок через `editingColumnId`
- Расширен тип пропа `additionalColumns` до `TaskListColumn<any>[]`
